# Setup
* curl -Lks http://bit.do/cfg-init | /bin/bash
* Install tmux plugins `https://github.com/tmux-plugins/tpm`
* `prefix` + <kbd>I</kbd> to install tmux plugins
* `prefix` + <kbd>r</kbd> to reload tmux 
* Install vim plugins `:PluginInstall`
